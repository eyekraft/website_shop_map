This module allows you to create a list of shops
to display on an interactive map.

Adaptive

Automatic geolocation at the address of the shop. Accuracy of geolocation of 30 meters

Informative

All contact data on the shop card. Shop work schedule by days of the week. Embedding photos of the shop

Flexible

Work beautifully on desktop and mobile devices. Usage of own snippet with Yandex map engine or Partner Google Map Snippet to show the map

Selective

Assign shop properties for dynamic or permanent filtering
